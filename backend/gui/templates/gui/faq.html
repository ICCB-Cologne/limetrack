FAQ

  1. Open a new file in excel.
  2. Click on the Data tab.
  3. Click Get External Data From Text.
  4. Choose Text/CSV as data source. Search your system for the CSV file. Select it, click Import and then Continue.
  5. Now the data is displayed as a preview. Click on the Load button and a new table is created on your excel sheet.

  1. Download the template file.
  2. Fill out the fields according to your group membership.
  3. Make sure that you meet the constraints on each field.
    Meaning: check the permitted inputs, format dates in the required style etc.
    You can take the example sample in the template as an reference on how the inputs are supposed to look.
  4. If you edit the file in Excel don't forget to export to CSV.
    How to do that? Click on File then on Save as ... and select the file format .csv with UTF-8 encoding before saving it.
  5. Now you can upload the file.

For the handling of special characters, uploaded files must be encoded in UTF-8.
UTF-8 is the dominant standard encoding on the web.
That's why you have to export your Microsoft Excel sheet to a CSV file with UTF-8 encoding.
