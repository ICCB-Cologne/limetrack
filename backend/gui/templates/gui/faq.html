  1. Open the Sample Tracking Tab.

  3. Fill out the fields according to your group membership.
  4. Make sure you meet the constraints on each field.
    Meaning: check the permitted inputs, format dates in the required style etc.

  2. Use the Filter columns button on the left to filter for columns according to your group membership.
  3. Download the filtered sample view table.
  4. Make sure that you meet the constraints on each field.

  5. Upload file in the upload section in the Sample Tracking Tab.

Filter out columns: Click the Filter Columns button to open a dropdown where you can select the columns displayed in the table.
You can filter out columns of the different groups involved in the SATURN3 Sample Tracking.
This makes it possible to e.g. only look at the data provided by the SPL.
Click the Apply button to apply.

Search filter: Use the Search section in the center above the table. Select a column you like to have searched. Then type in what you are searching for. The search is not case sensitive and gives out all occurrences of the input characters.
Example: Searching for "S3C-" in the "SATURN3 Sample Code" column will present you all the records whose SATURN3 Sample Codes include "S3C-".

Filter within individual columns: Click on the small arrow beside a column's heading.
Here you have the opportunity to select and deselect values you want or do not want to see in the table.
Example: If you only want male patients to appear in the table, you can hide the female patients' records by unchecking the "f" in column "Sex".
